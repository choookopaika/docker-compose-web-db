# Тестовое задание: "Docker Compose Challenge: Web + DB"
### Цель: Развернуть связку из веб-приложения и базы данных, обеспечив их взаимодействие и сохранность данных.
Легенда: Нужно поднять простейший сервис "счетчик посещений". Приложение должно принимать HTTP-запрос, записывать информацию о визите в базу данных и возвращать текущее количество записей.
## Запуск

В корне проекта выполнить команду:

```bash
docker-compose up -d --build
```
## Проверка работоспособности

После запуска выполнить:

```bash
curl http://localhost:8000
```

## Ожидаемый ответ:
```bash
Hello! I have been visited X times
```
Проверка проходила на ВМ kali.
